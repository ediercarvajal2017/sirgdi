EL PROBLEMA
═══════════════════════════════════════════════════════════════════════════

❌ Error que aparecía:
   "Error: Error ejecutando query. Contacte al administrador"
   Ubicación: lib/basedatos.php:70

❌ Síntomas:
   • Aparecía al iniciar sesión
   • Aparecía al crear reportes
   • Aparecía al gestionar técnicos
   • Aparecía en exportación de datos
   • SE REPETÍA constantemente

❌ Causa Raíz:
   El código estaba usando nombres de columnas ANTIGUOS que ya no 
   existían en la base de datos. Ejemplo:

   SELECT nombre FROM usuario
   ❌ "nombre" no existe (debe ser "nombre_completo")
   
   SELECT * FROM sla_config
   ❌ Tabla "sla_config" no existe (usar "sla")
