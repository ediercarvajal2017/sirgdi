📋 ¿QUÉ SE IMPLEMENTÓ?
═══════════════════════════════════════════════════════════════════════════

Un sistema profesional de notificaciones emergentes (Toasts) que muestra
feedback visual inmediato al usuario cuando:

  ✅ Guarda datos exitosamente
  ❌ Encuentra errores de validación
  ⚠️  Necesita confirmar una acción peligrosa
  ℹ️  Recibe información importante

🎨 4 TIPOS DE NOTIFICACIONES
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│ ✅ ÉXITO (Verde #27AE60)                                                │
│ └─ Usar para: Crear, guardar, eliminar, completar acciones             │
│ └─ Ejemplo: toast.success('¡Éxito!', 'Institución creada')            │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ ❌ ERROR (Rojo #E74C3C)                                                  │
│ └─ Usar para: Validación fallida, conexión perdida, permiso denegado   │
│ └─ Ejemplo: toast.error('Error', 'DANE ya está registrado')            │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ ⚠️  ADVERTENCIA (Naranja #F39C12)                                        │
│ └─ Usar para: Acciones irreversibles, límites, sesión expirando        │
│ └─ Ejemplo: toast.warning('Advertencia', 'Esta acción no se puede...')  │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ ℹ️  INFORMACIÓN (Azul #3498DB)                                           │
│ └─ Usar para: Notificaciones, tips, progreso                           │
│ └─ Ejemplo: toast.info('Info', 'Datos siendo procesados')              │
└─────────────────────────────────────────────────────────────────────────┘

💻 INTEGRACIÓN ACTUAL
═══════════════════════════════════════════════════════════════════════════

✅ COMPLETADO en controlador_superadmin.php:

  Cuando guardas una institución:
  ┌─────────────────────────────────────────────────────────────┐
  │ 1. Formulario se envía                                      │
  │ 2. Controlador procesa datos                                │
  │ 3. Si éxito: $_SESSION['exito'] = 'mensaje'               │
  │ 4. Si error:  $_SESSION['error'] = 'mensaje'               │
  │ 5. Redirecciona a la página                                 │
  │ 6. Toast aparece automáticamente ✨                         │
  │ 7. Usuario ve confirmación inmediata                        │
  └─────────────────────────────────────────────────────────────┘


📝 EJEMPLO DE USO
═══════════════════════════════════════════════════════════════════════════

En tu controlador:

  // Éxito
  $_SESSION['exito'] = 'Institución creada exitosamente';

  // Error
  $_SESSION['error'] = 'El código DANE ya está registrado';

  // Advertencia
  $_SESSION['advertencia'] = 'Esta acción no se puede deshacer';

  // Información
  $_SESSION['info'] = 'Los datos están siendo procesados';


O directamente en JavaScript:

  toast.success('¡Éxito!', 'Mensaje adicional');
  toast.error('Error', 'Algo salió mal');
  toast.warning('Advertencia', 'Verifica antes de continuar');
  toast.info('Información', 'Operación en progreso...');
